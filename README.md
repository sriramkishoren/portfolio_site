# Portfolio Site

Personal portfolio website for Sriramkishore Naraharisetti.

## Live Site

**[www.kishoretech.com](https://www.kishoretech.com)**

## Overview

A static portfolio website hosted on GitHub Pages featuring:

- Professional portfolio landing page
- Personal blog section with post viewer
- Dark mode support
- Responsive design with Tailwind CSS

## Development

No build process required. Open `index.html` directly in a browser or use any local server.

## Deployment

The site deploys automatically to GitHub Pages when changes are pushed to the `main` branch.

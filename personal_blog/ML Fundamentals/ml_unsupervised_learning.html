Unsupervised Learning

"Finding hidden patterns without a teacher"

Discovery Mode No Labels

What is it?

A class of machine learning techniques used to find patterns in unlabeled data. Unlike traditional prediction problems, the data doesn't know what it is. There are no dependent or independent variables.

Visual Metaphor

Imagine sorting a messy drawer of photos without knowing who anyone is. You just group "similar looking" people together. That's unsupervised learning.

Supervised vs. Unsupervised

The "Teacher"

Supervised

  • Has Labels ("This is a tiger")
  • Goal: Prediction
  • Input & Output are known
The "Explorer"

Unsupervised

  • No Labels (Raw data)
  • Goal: Discovery / Inference
  • Finds structure on its own

How It Works

Input Data

Images, text, or numbers without any tags or names.

Pattern Mining

Algorithm measures similarities and distances between points.

Structure Found

Data grouped into clusters or simplified rules.

Core Techniques

Clustering

Grouping data points so that objects in the same group are more similar to each other than to those in other groups.

Ex: Grouping customers by purchasing behavior

Dimensionality Reduction

Compressing data by reducing the number of random variables under consideration, keeping only important features.

Ex: Compressing images without losing quality

Association Rules

Discovering interesting relations between variables in large databases.

Ex: "People who buy bread also buy milk"

Anomaly Detection

Identifying rare items, events, or observations which raise suspicions by differing significantly from the majority.

Ex: Credit card fraud detection

The Algorithm Toolkit

K-Means

Partitioning n observations into k clusters.

DBSCAN

Density-based clustering. Finds outliers.

PCA

Principal Component Analysis. Reduces dimensions.

t-SNE

Visualizing high-dimensional data.

Apriori

Frequent item set mining for rules.

Hierarchical

Builds a tree of clusters (dendrogram).

Real-World Magic

Market Basket

Recommending products based on cart contents.

Segmentation

Grouping customers for targeted marketing.

Genetics

Grouping genes with similar expression patterns.

The Trade-off

Discover Hidden Insights Finds things you didn't know to look for.
Less Manual Work No need to manually label thousands of images.
Unpredictable You don't define the output, so results can be vague.

Key Terminology

Unlabeled Data Data without tags (e.g., photos without names).
Cluster A collection of data points that are similar to one another.
Centroid The center point of a cluster.
Noise Data points that don't fit into any cluster (outliers).

Types of Machine Learning

Machine learning empowers computers to learn from data without explicit programming. Based on how they learn, algorithms are categorized into three main types.

Supervised Learning

Definition

The algorithm learns from a labeled dataset, using "supervisory" input data to predict specific outputs. Itâ€™s like learning with a teacher who provides the correct answers key.

Real-World Example

Loan Default Prediction: A bank uses historical customer data (income, credit score) labeled with outcomes (defaulted/paid) to predict if a new applicant is risky.

When to use

When you have historical data with known outcomes (labels) and want to predict numbers (Regression) or categories (Classification).

Unsupervised Learning

Definition

The algorithm analyzes raw, unlabeled data to discover hidden patterns, structures, or groups on its own. There is no "teacher" or predefined answer key.

Real-World Example

Customer Segmentation: Grouping customers based on purchasing behavior (e.g., frequent small buyers vs. one-time luxury buyers) without predefined categories.

When to use

To simplify complex data (Dimension Reduction) or find natural groupings (Clustering) when you don't know the specific outcome you are looking for.

Reinforcement Learning

Definition

An agent learns to make decisions by interacting with an environment. It learns the "best action" through a system of trial and error, maximizing rewards and minimizing penalties.

Real-World Example

Self-Driving Cars: The car learns to stay in its lane and stop at red lights by receiving "rewards" for safe driving and "penalties" for rule violations.

When to use

When an automated agent needs to make a sequence of decisions in a dynamic environment to achieve a long-term goal (e.g., robotics, games).

How to Choose?

Have labeled data?
Yes → Supervised
Exploring patterns?
Yes → Unsupervised
Interacting with environment?
Yes → Reinforcement

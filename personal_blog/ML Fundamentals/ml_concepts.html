Core ML Concepts

A visual guide to the four pillars of machine learning: predicting values, sorting categories, simplifying data, and discovering groups.

Regression

Predicting

A supervised learning technique used to predict a continuous numerical value based on the relationship between variables.

Output: Continuous Value
Metaphor Like looking at a child's height chart to guess how tall they will be next year.
Real-World Example Predicting house prices based on square footage and location.

Input: Historical Data → Output: Number

Classification

Sorting

A supervised learning method where the model learns to assign specific labels or categories to new input data.

Output: Category Label
Metaphor Sorting your mail into "Bills", "Letters", and "Junk".
Real-World Example Determining if a medical tumor is benign or malignant.

Input: Labeled Data → Output: Label

Dimension Reduction

Simplifying

The process of reducing the number of variables in data while preserving the most important information.

Output: Simplified Features
Metaphor Writing a synopsis of a long book. You lose some details, but get the main plot.
Real-World Example Compressing large image files (JPG) without losing visible quality.

Input: Complex Data → Output: Key Features

Clustering

Grouping

An unsupervised learning technique that groups similar data points together without pre-existing labels.

Output: Groups
Metaphor Organizing a mixed library of books by genre without checking the covers first.
Real-World Example Customer segmentation (grouping shoppers by purchasing habits).

Input: Messy Data → Output: Structured Groups
